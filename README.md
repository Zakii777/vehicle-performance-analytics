# Vehicle Performance Analytics

## Overview

This project analyzes vehicle performance data and applies machine learning techniques to predict fuel efficiency (MPG) and classify vehicle types.

## Tools Used

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-Learn

## Workflow

- Data Cleaning
- Missing Value Treatment
- Exploratory Data Analysis
- Feature Engineering
- Regression Modeling
- Classification Modeling

## Results

### Regression Model (MPG Prediction)

- Training R²: 0.822
- Testing R²: 0.829

### Classification Model (Vehicle Type)

- Training Accuracy: 86.16%
- Testing Accuracy: 81.25%

## Key Findings

- Vehicle characteristics strongly influence fuel efficiency.
- Machine learning models can accurately predict MPG.
- KNN achieved over 81% classification accuracy on unseen data.
